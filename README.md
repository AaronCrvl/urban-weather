# ☁️Urban Weather
Simple weather app made with Next Js.

## 🧪 Stack

- [Next JS](https://nextjs.org/)    
- [Typescript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Atom - Jotai(State Management)](https://jotai.org/docs/core/atom)

## 💤 Starting The App
``` javascript
npm run dev
```
